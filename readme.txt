Adding a photo vote to the WordPress Gallery

== Description ==
Adding a photo vote to the WordPress Gallery.

= Features =

* Protection against cheating votes by IP, cookies
* Setting the number of votes from 1 person
* Adding styles for responsive gallery
* Easy to use

== Screenshots ==

1. Voting settings in WP standart gallery
2. How it looks on the site

== Installation ==

1. Unzip the downloaded zip file.
1. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
1. Activate `Voting for a photo` from Plugins page
